Description
===========

A collection of DANS XML schemas. They mainly serve to support the [DANS BagIt Profile v1]{:target=_blank}. To avoid dependencies on thirdparty sites hosting 
referenced schemas, copies of those schemas have been incorporated. Note, that this site may also include several legacy schemas carried over from the old
[EASY]{:target=_blank} archival system. 

[DANS BagIt Profile v1]: https://dans-knaw.github.io/dans-bagit-profile/versions/1.2.0/
